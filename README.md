## 🛠️ Requisitos Previos

Antes de clonar y ejecutar el proyecto, asegúrate de tener instalado en tu sistema:

*   **Linux:** Docker y Docker Compose instalados de manera nativa.
*   **Windows:** 
    *   [Docker Desktop](https://www.docker.com/products/docker-desktop/) (Asegúrate de activar la casilla **"Use the WSL 2 based engine"** durante la instalación).
    *   Una distribución de [WSL 2](https://learn.microsoft.com/es-es/windows/wsl/install) (Recomendado: Ubuntu).
    *   **Importante (Windows):** Debes clonar este repositorio **dentro de tu terminal de WSL** (por ejemplo en `/home/usuario/proyectos`), nunca en el disco local de Windows (`/mnt/c/`), o irá extremadamente lento.
